# ViralCaption

A modern, responsive SaaS-style web application for generating viral TikTok captions instantly.

## Features

- **Instant Caption Generation**: Choose from 7 different moods and content categories
- **Mobile-First Design**: Fully responsive design that works perfectly on all devices
- **Dark Theme UI**: Modern AI-tool inspired interface
- **Local Dataset**: 70+ pre-optimized captions for maximum engagement
- **Copy to Clipboard**: One-click caption copying
- **Category Selection**: Click category cards to auto-select in generator
- **Email Waitlist**: Join early access for future features

## Tech Stack

- **HTML5**: Semantic markup structure
- **Tailwind CSS**: Utility-first CSS framework (CDN)
- **Vanilla JavaScript**: Client-side functionality
- **Inter Font**: Modern typography

## Getting Started

1. **Clone or download** the project files
2. **Open `index.html`** in your web browser
3. **Start generating** viral captions!

No build process or server required - everything runs client-side.

## Usage

1. **Select Mood**: Choose from Motivational, Funny, Romantic, Business, Storytelling, POV, or Lifestyle
2. **Select Category**: Pick a content category (Success, Love, Hustle, etc.)
3. **Generate**: Click "Generate Caption" to get a viral-ready caption
4. **Copy**: Use the copy button to copy the caption to your clipboard
5. **Generate Another**: Get a different caption instantly

## Caption Categories

The app includes 70+ captions across 7 categories:

- **Motivational**: Inspiring and uplifting content
- **Funny**: Humorous and relatable captions
- **Romantic**: Love and relationship focused
- **Business**: Professional and entrepreneurial
- **Storytelling**: Narrative and journey-based
- **POV**: First-person perspective captions
- **Lifestyle**: Daily life and wellness content
